Important implementation notes:
1. The contact form is a static front-end demo. Connect it to WordPress, an email service, or CRM before launch.
2. Replace SVG placeholder visuals with real product photos, factory photos, catalog PDFs and verified product specification tables.
3. Keep model specifications verified before publishing. Avoid using unconfirmed data.
4. Header and footer are duplicated across static HTML files for easy standalone deployment.
